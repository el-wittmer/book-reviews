# Book Journal 
## Objective
The goal of this project is to advance past serving on localhost or github pages to create a database-focused web application that runs on a remote server.

- [x] Deploy on AWS
- [x] Add CRUD functionality
- [ ] Hide Create, Update, Delete functionality behind authentication

## Tech Stack
This project was written using EJS templating, making it a mix of HTML and Javascript. Routing is done through Express. I chose to use PostgreSQL to create the database and the application is deployed on an AWS EC2 instance.
